01. 파일 저장시 에러가 발생하는 경우
    1.USB와 같은 외장 장치에 폴더를 생성하는 경우

02. 소스 제어에서 리포지토리 초기화 버튼이 없는 경우
    1. 설정된 이메일 주소 재확인 
        - 나의 경우 터미널을 통해 설정했던 이메일 주소의 오류가 있었고,
        이메일 주소를 수정하여 해결이 됨
        * 이메일 주소는 Git hub에 가입된 계정과 같아야 함
    2. 비주얼 스튜디오 계정 확인
        - 1번과 연관

03. 커밋 실행시 반드시 버전별 이름을 입력해야 하고 그 이름은 동일 하면 안됨
    - 예 : 첫번째 커밋, 두번째 커밋, 세번째 커밋
    - 예 : ver0.1, ver0.2, ver1, ver2

04. 변경될 또는 변경된 파일은 저장후 스테이징 시켜야함
    - 소스 제어 패널에서 파일에 마우스 오버시 나오는 + 버튼 클릭

05. 최초 브런치 설정시 public 버전을 선택하여야 함

06. 폴더 생성시 한글이 들어가는 경우 Git hub에서 null로 표시되는 것을 발견, 폴더명은 영어로 해야함.
    - 파일명은 한글이 들어가도 무방

07. 소스제어 패널에서 커밋 실행바가 계속 움직이고 시간이 많이 지나도 움직이는 경우 작업중인 모든 파일은 저장을 하고 비쥬얼 스튜디오 프로그램을 껏다가 다시 켜볼 것.
    - 나의 경우 이방법으로 해결됨
